## 使用示例

### 1. 查询股票代码

```python
from app.services.market_query import MarketQueryService
from app.utils.database import SessionLocal

db = SessionLocal()
service = MarketQueryService(db)

# 查询招商银行
results = service.search_by_name('招商银行', asset_type='stock', limit=5)
print(results[0])
# 输出：{'code': '600036', 'name': '招商银行', 'market': '上海证券交易所'}
```

### 2. 获取市场信息

```python
# 获取详细信息
info = service.get_market_info_by_code('600036')
print(info)
```

### 3. 同步市场数据

```python
# 同步单个股票数据
success = service.sync_market_data('600036')
if success:
    # 再次查询就有价格数据了
    info = service.get_market_info_by_code('600036')
    print("收盘价:", info.get('close'))
```
