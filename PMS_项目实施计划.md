# 投资组合管理系统 (PMS) 项目实施计划

## 1. 项目概述

本项目旨在开发一个功能完备、用户友好的投资组合管理系统，为个人和机构投资者提供投资组合管理、资产分析、业绩评估和市场数据等核心功能。系统采用前后端分离架构，前端使用 Vue 3 + TypeScript，后端使用 Python + Flask，数据库使用 SQLite。
