# PMS 功能规划：投资组合与资产关系

## 需求分析

根据需求规范文档：
- 每个投资组合都归属于特定用户
- 每个投资组合拥有0..n个资产
- 资产可以是股票、基金、债券、现金

## 关系说明

```
用户 (User)
  │
  └── 1:N 投资组合 (Portfolio)
            │
            └── 1:N 持仓 (Holding)
                      │
                      └── N:1 资产 (Asset)
                                │
                                └── type: 股票/基金/债券/现金
```

## 注意事项

- 现金资产类型处理方式不同于股票/基金，不需要实时价格
- 每个资产只能属于一个投资组合的持仓（通过 holdings 表关联）
- 资产信息全局共享，但持仓记录按投资组合隔离
