## 命名规范

### 文件命名

- 全部使用小写字母
- 多个单词使用下划线分隔
- 示例：`portfolio.py`, `user_service.py`

### 类命名

- 使用 PascalCase（大驼峰）
- 示例：`Portfolio`, `UserService`

```python
class Portfolio(Base):
    __tablename__ = 'portfolios'
```

### 函数和变量命名

- 使用 snake_case（小写 + 下划线）
- 私有方法/变量使用单下划线前缀

```python

def _calculate_metrics(data: dict) -> dict:
    """内部方法"""
    pass
```

### 常量命名

- 全部大写，多个单词使用下划线分隔

```python
ASSET_TYPES = ['stock', 'fund', 'bond', 'cash']
```

### 数据验证

使用 Marshmallow schema 进行数据验证：

```python
from marshmallow import Schema, fields, validate

class PortfolioCreate(Schema):
    """投资组合创建 Schema"""
    name = fields.String(
        required=True, 
        validate=validate.Length(min=1, max=100)
    )
    description = fields.String(
        validate=validate.Length(max=500)
    )
    benchmark = fields.String(
        required=True,
        validate=validate.Length(max=50)
    )
    risk_level = fields.String(
        required=True,
        validate=validate.OneOf(['低风险', '中风险', '高风险'])
    )
    is_default = fields.Boolean(load_default=False)
```

## 数据库规范

### 模型定义

```python
from datetime import datetime
from sqlalchemy import Column, Integer, String, DateTime, ForeignKey, Boolean
from sqlalchemy.orm import relationship
from app.utils.database import Base

class Portfolio(Base):
    """投资组合模型"""
    
    __tablename__ = 'portfolios'
    
    id = Column(Integer, primary_key=True, index=True)
    user_id = Column(Integer, ForeignKey('users.id'), nullable=False, index=True)
    name = Column(String(100), nullable=False)
    description = Column(Text)
    benchmark = Column(String(50), nullable=False)
    risk_level = Column(String(20), nullable=False)
    is_default = Column(Boolean, nullable=False, default=False)
    created_at = Column(DateTime, nullable=False, default=datetime.utcnow)
    updated_at = Column(DateTime, nullable=False, default=datetime.utcnow, onupdate=datetime.utcnow)
    
    # 关系
    user = relationship('User', back_populates='portfolios')
    holdings = relationship('Holding', back_populates='portfolio', cascade='all, delete-orphan')
```

## 服务层规范

### 业务逻辑封装

```python
from typing import List, Optional
from sqlalchemy.orm import Session
from app.models.portfolio import Portfolio
from app.schemas.portfolio import PortfolioCreate

class PortfolioService:
    """投资组合服务"""
    
    def __init__(self, db: Session):
        self.db = db
    
    def get_portfolios(self, user_id: int) -> List[Portfolio]:
        """获取用户的投资组合列表"""
        return self.db.query(Portfolio).filter(
            Portfolio.user_id == user_id
        ).all()
    
    def create_portfolio(self, data: dict, user_id: int) -> Portfolio:
        """创建投资组合"""
        # 检查是否已有默认组合
        existing_default = self.db.query(Portfolio).filter(
            Portfolio.user_id == user_id,
            Portfolio.is_default == True
        ).first()
        
        # 如果是第一个组合或指定为默认，设置 is_default
        is_default = data.get('is_default', False)
        if not existing_default or is_default:
            if existing_default and is_default:
                # 取消其他组合的默认状态
                existing_default.is_default = False
        
        db_portfolio = Portfolio(
            user_id=user_id,
            name=data['name'],
            description=data.get('description'),
            benchmark=data['benchmark'],
            risk_level=data['risk_level'],
            is_default=is_default
        )
        
        self.db.add(db_portfolio)
        self.db.commit()
        self.db.refresh(db_portfolio)
        
        return db_portfolio
```

```python
from werkzeug.security import generate_password_hash, check_password_hash

class User(Base):
```
