# 市场查询服务使用指南

## 📋 概述

市场查询服务（`MarketQueryService`）提供根据股票或基金名称查询市场信息的功能，支持：

- ✅ 按名称搜索（支持部分匹配）
- ✅ 按代码查询
- ✅ 实时行情获取
- ✅ 多数据源集成（新浪、腾讯）
- ✅ 本地数据库缓存
- ✅ 批量数据同步

## 🏗️ 架构设计

### 类结构

```
MarketQueryService
├── 核心查询方法
│   ├── search_by_name()           # 按名称搜索
│   ├── get_market_info_by_code()  # 按代码查询
│   └── get_market_info_by_name()  # 按名称查询（精确/模糊）
├── 本地数据库搜索
│   ├── _search_local_database()   # 本地搜索
│   ├── _get_latest_market_data()  # 获取最新数据
│   └── _build_market_info()       # 构建信息对象
├── 外部 API 集成
│   ├── _search_external_api()     # 外部搜索
│   ├── _search_sina_stock()       # 新浪股票搜索
│   ├── _search_sina_fund()        # 新浪基金搜索
│   ├── _fetch_sina_market_data()  # 新浪行情
│   └── _fetch_tencent_market_data() # 腾讯行情
└── 数据同步
    ├── sync_market_data()         # 同步单个资产
    └── sync_all_assets()          # 批量同步
```

### 数据源

1. **本地数据库** - 优先使用，速度快
2. **新浪财经 API** - 实时行情，支持股票/基金
3. **腾讯财经 API** - 备用数据源

## 📖 使用方法

### 1. 基础查询

#### 按名称搜索

```python
from app.services.market_query import MarketQueryService
from app.utils.database import SessionLocal

db = SessionLocal()
service = MarketQueryService(db)

# 搜索名称包含"平安"的股票
results = service.search_by_name('平安', asset_type='stock', limit=10)

for result in results:
    print(f"代码：{result['code']}, 名称：{result['name']}, 价格：{result['price']}")

db.close()
```

#### 按代码查询

```python
# 获取贵州茅台的市场信息
info = service.get_market_info_by_code('600519')

if info:
    print(f"名称：{info['name']}")
    print(f"当前价：{info['price']}")
    print(f"涨跌额：{info['change']}")
    print(f"涨跌幅：{info['change_percent']}%")
```

#### 按名称查询

```python
# 模糊匹配（推荐）
info = service.get_market_info_by_name('茅台')

# 精确匹配
info = service.get_market_info_by_name('贵州茅台', exact=True)
```

### 2. 便捷函数

```python
from app.services.market_query import (
    search_stock_by_name,
    search_fund_by_name,
    get_market_info
)

db = SessionLocal()

# 搜索股票
stocks = search_stock_by_name('平安银行', db, limit=10)

# 搜索基金
funds = search_fund_by_name('沪深 300', db, limit=10)

# 获取市场信息
info = get_market_info('600519', db)

db.close()
```

### 3. 数据同步

#### 同步单个资产

```python
# 同步贵州茅台的市场数据
success = service.sync_market_data('600519')

if success:
    print("同步成功！")
else:
    print("同步失败")
```

#### 批量同步

```python
# 同步所有股票和基金
success_count, fail_count = service.sync_all_assets()

print(f"成功：{success_count}, 失败：{fail_count}")
```

## 🔍 搜索功能详解

### 搜索策略

1. **本地优先** - 先在本地数据库搜索，速度快
2. **外部补充** - 本地结果不足时，调用外部 API
3. **智能去重** - 自动去重本地和外部结果

### 匹配规则

- **名称匹配** - 支持拼音、部分名称
- **代码匹配** - 支持完整或部分代码
- **类型筛选** - 可按 stock/fund/bond 筛选

### 示例

```python
# 搜索所有包含"茅台"的资产
service.search_by_name('茅台')

# 只搜索股票
service.search_by_name('茅台', asset_type='stock')

# 搜索代码包含"600"的股票
service.search_by_name('600', asset_type='stock')

# 搜索基金
service.search_by_name('沪深 300', asset_type='fund')
```

## 🛠️ 高级功能

### 1. 自定义数据源

可以扩展 `MarketQueryService` 添加自定义数据源：

```python
class CustomMarketQueryService(MarketQueryService):
    def _fetch_custom_market_data(self, code: str):
        """自定义数据源"""
        # 实现你的数据获取逻辑
        pass
    
    def _fetch_from_external_api(self, code: str):
        """重写外部 API 获取方法"""
        # 优先使用自定义数据源
        data = self._fetch_custom_market_data(code)
        if data:
            return data
        
        # 回退到默认数据源
        return super()._fetch_from_external_api(code)
```

### 2. 缓存机制

添加 Redis 缓存提高性能：

```python
import redis
import json

class CachedMarketQueryService(MarketQueryService):
    def __init__(self, db: Session, redis_client: redis.Redis):
        super().__init__(db)
        self.redis = redis_client
    
    def get_market_info_by_code(self, code: str):
        # 尝试从缓存获取
        cached = self.redis.get(f'market:{code}')
        if cached:
            return json.loads(cached)
        
        # 缓存未命中，从数据库或 API 获取
        info = super().get_market_info_by_code(code)
        
        # 存入缓存（5 分钟过期）
        if info:
            self.redis.setex(f'market:{code}', 300, json.dumps(info))
        
        return info
```

### 3. 异步支持

使用 asyncio 提高并发性能：

```python
import asyncio
import aiohttp

class AsyncMarketQueryService:
    async def fetch_market_data(self, code: str):
        """异步获取市场数据"""
        async with aiohttp.ClientSession() as session:
            url = f"http://hq.sinajs.cn/list={code}"
            async with session.get(url) as response:
                data = await response.text()
                return self._parse_data(data)
    
    async def fetch_multiple(self, codes: List[str]):
        """批量异步获取"""
        tasks = [self.fetch_market_data(code) for code in codes]
        results = await asyncio.gather(*tasks)
        return results
```

## ⚠️ 注意事项

### 1. API 限制

- 新浪/腾讯 API 有访问频率限制
- 建议添加请求间隔（0.5-1 秒）
- 生产环境建议使用付费数据服务

### 2. 错误处理

```python
try:
    info = service.get_market_info_by_code('600519')
    if info:
        print(info)
    else:
        print("未找到该资产")
except Exception as e:
    print(f"查询失败：{e}")
```

### 3. 数据准确性

- 实时行情有 15 分钟延迟
- 收盘价以交易所为准
- 重要决策请核实多个数据源
